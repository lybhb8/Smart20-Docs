OMRON CX-ONE4
==============
CX-One是一种综合性软件包 , 为PLC编程软件集成了支持软件, 可对包括网络、可编程终端、伺服系统、变频器、电子温度控制器进行设置。

* `OMRON CP/CJ系列PLC手册 <https://www.aliyundrive.com/s/JqTns18FwgS>`_
